## 🚀 Özellikler
- ✅ Otomatik veri çıkarımı (PDF, JPG, TXT, DOCX, XLSX)
- ✅ Kişisel veri anonimleştirme
- ✅ Dinamik Excel şablonları
- ✅ Otomatik kümelendirme ve raporlama
- ✅ API desteği ve Web arayüzü
