# 🧠 BotExcel

**BotExcel**, PDF, görsel (JPG, PNG) ve düz metin dosyalarındaki verileri yapay zeka (OCR + işleme) kullanarak Excel'e otomatik olarak dönüştüren, yerel çalışan ve açık kaynaklı bir araçtır.

## 🚀 Özellikler
- ✅ PDF metin çıkarımı (PyMuPDF)
- ✅ Görselden OCR (Tesseract)
- ✅ Metin dosyası desteği
- ✅ Excel çıktısı (openpyxl)
- ✅ Komut satırı ve Web arayüzü desteği (Flask)
